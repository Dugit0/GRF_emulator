GRF_emulator

GRF emulator - интерпретатор языка, который основан на частично рекурсивных функциях и только на них.

Запуск

Запуск производится командами

pipenv shell python -m grfemulator

Синтаксис языка

Функции

Тождественный ноль

o - одноместный тождественный 0, т.е. o(x)=0.

Пример:

o

Функция следования

s - функция следования, т.е. s(x)=x+1.

Пример:

s

Функция выбора

I^<n>_<m> - функция выбора с параметрами n и m, т.е.

Imn(x1,…,xn)=xm

.

Пример:

I^5_2

Который обозначает:

I25(x1,…,x5)=x2

Константные функции

  • 0^<n> - n-местный тождественный 0, т.е. on(x1,…,xn).
  • <const>^<n> - n-местная тождественная константа С, т.е. s(s(...s(on(x1,…,xn))...)).

Примеры:

0^3 42^1 150^123

Которые обозначают функции

o(x1,x2,x3),42(x1),150(x1,…,x123)

Операции над функциями

Суперпозиция

F {A, B, C, ..., Z} - суперпозиция S(F,A,B,C,…,Z).

Пример:

U {X, Y, Z}

Который обозначает:

S(U,X,Y,Z)

Примитивная рекурсия

G <- H - примитивная рекурсия R(G,H).

Минимизация

? F - минимизация μ(F).

Создание новых функций

Возможно создание пользовательских функций с именами, которые возможны по правилам языка Си.

F = s; G = I^5_1;

Которые обозначают функции

F(x)=s(x)
G(x1,x2,x3,x4,x5)=I15(x1,x2,x3,x4,x5)

Вызов функций

Вызов функций производится в отдельном окне вызова функций, которое по умолчанию расположено в правом верхнем углу. Синтаксис вызова функций:

F(1, 2, 3, 10)
