GRF_emulator

GRF emulator - интерпретатор языка, который основан на частично рекурсивных функциях и только на них.

Запуск

Запуск производится командами

pipenv shell
python -m grfemulator

Синтаксис языка

Функции

Тождественный ноль

o - одноместный тождественный 0, т.е. .

Пример:

o

Функция следования

s - функция следования, т.е. .

Пример:

s

Функция выбора

I^<n>_<m> - функция выбора с параметрами и , т.е. .
